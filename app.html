Peaceful Vanilla Club App

Downloading

Click the button below. A .zip archive will download, which can be extracted to reveal multiple files. (We suggest you extract into a folder). Run the main executable and the app will run :D

Using

Simply download, run the installer and open the app. The links across the top are the main feature, allowing for quick access between sites such as vote links.

Editing

Feel free to edit the source code. The main.js file contains almost all the main logic, in which you can change the menus, and even load up entirely new webpages!
